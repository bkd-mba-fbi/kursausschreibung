Settings

Settings

Die genau Dokumentation finden Sie im Wiki auf GitHub.

  • Anzuzeigende Veranstalter sind konfigurierbar
  • Das Paging für die Anzahl Anlässe pro Seite ist konfigurierbar
  • Für die Anlassliste/-details sind der Titel und die angezeigten Felder konfigurierbar
  • Anlasstexte können ein- und ausgeblendet werden. Formatierungen aus Evento NG werden auch in HTML gleich dargestellt
  • Batch einblendbar Anzahl Freiplätze
  • Personendetail: Pro «Anlasstyp» kann gesteuert werden welche Personendaten Felder angezeigt werden und welche Pflicht sind
  • Die Sprache der Navigation kann ausgeschaltet werden
  • Lektionen eines Anlass als Kalendereintrag exprtieren
  • Das Modul ist zweisprachig (de-CH, fr-CH): die Texte sind frei übersetzbar
  • Das Ampelsystem für die Anmeldung auf publizierte Anlässe lässt sich individuell einstellen
  • Das Logo mit Link lässt sich im rechten Grid einblenden
  • Kontakt Card kann eigens mit HTML konfiguriert werden
  • Die Twitter Timeline kann eingeblendet werden
